Globaladtool Tool

Globaladtool is a Moodle tool that allows add or remove Globalad block to all courses into categorie and into Dashboard to all users. The block can be placed whenever the Theme allows it, in the region of the left or right and up or botton position of this.

Globaladtool es una herramienta para Moodle que permite añadir o quitar el bloque Globalad a todos los cursos de una categoría y en el Área personal de todos los usuarios. El bloque podemos colocarlo siempre que el Tema lo permita, en la región de la izquierda o derecha y en la posición superior o inferior de esta.



Installation

Globaladtool should be installed like any other Moodle tool. See https://docs.moodle.org/36/en/Installing_plugins


Dependencies

The Globalad block (2019011706 or higher) must be installed previously. See https://moodle.org/plugins/pluginversions.php?plugin=block_globalad
